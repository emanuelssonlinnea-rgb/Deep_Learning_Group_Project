# Utforskning och fördjupning i neurala nätverk

## Introduktion
Projektet går ut på att bygga en klassificeringsmodell som kan förutsäga vilken kategori ett klädesplagg tillhör baserat på en bild.
Det innebär att modellen ska:
- ta en bild som input
- analysera innehållet
- förutsäga rätt klass (t.ex. sko, tröja, väska)
Målet är inte bara att få hög accuracy, utan att förstå hur olika val i modellen påverkar resultatet.
